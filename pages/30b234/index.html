算數運算子

# Java中的算數運算子

在 Java 中，基本的運算總共有五種，除了我們常見的加減乘除以外還多了一個模數運算

算術運算子 使用範例 說明
+ a + b a 加上 b
- a - b a 減掉 b
* a * b a 乘上 b
/ a / b a 除以 b
% a % b 取 a 除以 b 的餘數

提示

整數除法的時候，因為整數型態的緣故並不會有小數點後的數字(可以理解為小數點後會捨去)

注意

模數運算子(%)只能用在整數型態，如果用在非整數型態將會造成編譯錯誤

# 計算優先權

就如同數學中的四則運算一樣，在程式中的算術運算子也運算上的優先順序。

基本上可以理解為 小括號 優先於 乘除模數 優先於 加減法，接著便是 由左至右

詳細的優先級可見下表(1為最優先 2次之 以此類推)

運算子 優先值 相同優先值的運作順序
括號()、[] 1 右至左
遞增遞減 ++ -- 、 負號 - 、 NOT ! 、 補數 ~ 2 左至右
乘 * 、除 / 、模數 % 3 左至右
加減 + - 4 左至右
位元移動 << >> 無正負性位元右移>>> 5 左至右
大小於判斷 < > <= >= 6 左至右
等於判斷 == != 7 左至右
AND & 8 左至右
XOR ^ 9 左至右
OR ! 10 左至右
簡化AND && 11 左至右
簡化OR || 12 左至右
三元運算 ?: 13 右至左
指定運算 = 14 右至左
簡化指定運算 += -= *= /= %= &= |= ^= 15 右至左

# 範例

public class Main {
  public static void main(String[] args) {
    int a = 5, b = 3;
    int c;
    c = 5 % 3;// 計算後結果存回c變數中
    // c 為 2
  }
}
